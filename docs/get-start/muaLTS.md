# MUA LTS分支

## 1. 介绍

MUA LTS是Little Unicorn Bot为高校MC联盟（MUA）开设的专属分支，长期维护。该分支删去了main.py中SJTU相关的插件，便于MUA成员部署。

## 2. 切换到MUA LTS分支

在git clone以后，只需要输入以下命令即可切换到MUA LTS分支

```bash
git switch muaLTS
```
