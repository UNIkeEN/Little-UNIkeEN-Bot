## 1. 插件简介

| 插件名称 | 父类 | 触发关键词 | 触发权限 | 内容 |
| ---- | ---- | ---- | ---- | ---- |
| MorningGreet | StandardPlugin | '早安' | None | 获取bot的早上问好 |
| NightGreet | StandardPlugin | '晚安' | None | 获取bot的晚上问好 |

## 2. 示范样例

代码部分：

```python
ROOT_ADMIN_ID = [111, # 用户A  
                 222, # 用户B
```

```python
class MorningGreet(StandardPlugin):
    def judgeTrigger(self, msg:str, data:Any) -> bool:
        return msg.startswith('早安')
    def executeEvent(self, msg:str, data:Any) -> Union[None, str]:
        target = data['group_id'] if data['message_type']=='group' else data['user_id']
        send(target, random.choice(GOODMORNING_LIST), data['message_type'])
        return "OK"
    def getPluginInfo(self, )->Any:
        return {
            'name': 'MorningGreet',
            'description': '早安',
            'commandDescription': '早安',
            'usePlace': ['group', 'private', ],
            'showInHelp': True,
            'pluginConfigTableNames': [],
            'version': '1.0.0',
            'author': 'Unicorn',
        }
```
